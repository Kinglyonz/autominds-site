# Autominds Brand Guidelines

## Logo
- **Primary Mark:** Neural hexagon (logo.svg)
- **Usage:** Favicon, social, print
- **Minimum Size:** 24px

## Colors

### Primary Palette
| Name | Hex | Usage |
|------|-----|-------|
| **Violet** | `#8B5CF6` | Primary accent, CTAs, highlights |
| **Blue** | `#3B82F6` | Secondary accent, gradients |
| **Deep Black** | `#0A0A0F` | Backgrounds |
| **Dark Gray** | `#1A1A2E` | Cards, sections |
| **Light Gray** | `#94A3B8` | Body text |
| **White** | `#F8FAFC` | Headlines, emphasis |

### Gradient
```css
background: linear-gradient(135deg, #8B5CF6 0%, #3B82F6 100%);
```

## Typography

### Primary Font: Inter
- **Headlines:** Inter Bold (700)
- **Subheadlines:** Inter Semi-Bold (600)
- **Body:** Inter Regular (400)
- **Fallback:** system-ui, -apple-system, sans-serif

### Scale
| Element | Size | Weight |
|---------|------|--------|
| H1 | 56px | 700 |
| H2 | 40px | 600 |
| H3 | 24px | 600 |
| Body | 18px | 400 |
| Small | 14px | 400 |
